API Endpoints links:

  1. View all CUSTOMERS
  2. View all PRODUCTS
  3. View all ORDERS

             

API Endpoints List:

TYPE ENDPOINT DESCRIPTION
GET https://node-simple-shop.herokuapp.com/customers View all CUSTOMERS
POST https://node-simple-shop.herokuapp.com/customers Create new CUSTOMER
PUT https://node-simple-shop.herokuapp.com/customers/id Update CUSTOMER matching ID provided
DEL https://node-simple-shop.herokuapp.com/customers/id Delete CUSTOMER matching ID provided
GET https://node-simple-shop.herokuapp.com/products View all PRODUCTS
POST https://node-simple-shop.herokuapp.com/products Create new PRODUCT
PUT https://node-simple-shop.herokuapp.com/products/id Update PRODUCT matching ID provided
DEL https://node-simple-shop.herokuapp.com/products/id Delete PRODUCT matching ID provided
GET https://node-simple-shop.herokuapp.com/orders View all ORDERS
POST https://node-simple-shop.herokuapp.com/orders Create new ORDER
